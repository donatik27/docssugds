## `sqd auth`

Log in to the Cloud

```
USAGE
  $ sqd auth -k <value> [--interactive]

FLAGS
  -k, --key=<value>       (required) Cloud auth key. Log in to https://app.subsquid.io to create or update your key.
      --[no-]interactive  Disable interactive mode
```
